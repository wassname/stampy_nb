# stampy_nb

A debuggable RAG, with these features

  - rephrased
  - example answer

- [ ] TODO: add reranking
- citations, with tooltips and references

## Install requirements

This project uses [poetry](https://python-poetry.org/) for requirement and is set up for torch using cuda.
~~~
poetry install
~~~
